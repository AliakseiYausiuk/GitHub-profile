# Navigating the GitHub repository  
======================

## Описание 
======================

Двухстраничный сайт написаный на React. Реализовано добавление и удаление в LocalStorage запросов клиента.
Работа с GitHub Api.

## Технологии

1. TypeScript
2. Redux 
3. Toolkit
4. RTKQuery
5. Tailwind
